 HTTP и HTTPS - это протоколы передачи гипертекста, предназначенные для обмена данными на сайтах.
Протокол HTTPS необходимо использовать во всех случаях, когда на сайтах доступны онлайн-платежи и передача
конфиденциальных сведений, в противном случае их могут получить злоумышленники. 
 HTTPS - это расширение протокола HTTP, которое шифрует данные между вашим сайтом и браузером посетителя 
в целях предотвращения просмотра третьими лицами данных.

  HTTP - это протокол, в  котором используется технология "Клиен - Сервер". В http иформация передается
от клиента к серверу и обратно в незашифрованном виде, чем могут воспользоваться мошенники.
  HTTPS - это протокол, в котором используется шифрование данных с помощью криптографических протоколов - SSL и TLS.
Фактически это тот же протокол передачи данных, но с добавлением надстройки шифрования. Ключи шифрования 
хранятся на сервере, для расшифровки используется специальный SSL-сертификат.
